# Ενημερώσεις — Euroleague Guru

Σύνοψη των αλλαγών πάνω στην αρχική έκδοση `eurodraft_9.html` (ανάπτυξη: j.tsouros με Claude Code).
Η έκδοση που ανεβαίνει online είναι πλέον το **`index.html`** — ίδιο παιχνίδι, αλλά με backend για πραγματικό multiplayer.

## 🏀 Δεδομένα παικτών
- Πηγή: **επίσημο EuroLeague API** (στατιστικά ανά σεζόν) + **θέσεις & σωματικά στοιχεία από Wikidata**. Καλύπτει σεζόν **2005-06 έως 2025-26**.
- Κάθε παίκτης έχει πλέον **θέση** (PG/SG/SF/PF/C) και **αξία = PIR** (ο δείκτης απόδοσης της EuroLeague).
- **Μία κάρτα ανά παίκτη / εποχή / ομάδα**, με μέσο όρο σταθμισμένο στους αγώνες (αντί για πολλές κάρτες, μία για κάθε σεζόν).
- Προστέθηκε η σεζόν **2025-26** — νέες ομάδες: **Dubai Basketball** και **Hapoel Tel Aviv**.
- Παίκτες με **πολλαπλές θέσεις** (π.χ. Ward = SG/SF) εμφανίζονται σε **όλες** τις θέσεις που παίζουν.

## 🎮 Μηχανισμοί παιχνιδιού
- **Draft ανά θέση**: οι 6 επιλογές είναι **PG → SG → SF → PF → C → FLEX** (το FLEX δέχεται οποιαδήποτε θέση). Το spin της ομάδας περιορίζεται σε ομάδες που έχουν παίκτη για τη ζητούμενη θέση (χωρίς αδιέξοδα).
- **PIR ως 7η κατηγορία** στη βαθμολογία (μαζί με πόντους / ριμπάουντ / ασίστ / κλεψίματα / μπλοκ / λάθη).
- **Re-spin εποχής ΚΑΙ ομάδας**, μία φορά το καθένα ανά παιχνίδι.

## 🌐 Multiplayer (backend)
- Πραγματικό **persistence & multiplayer** με **Vercel KV (Redis)**: τα τουρνουά αποθηκεύονται στον server και παίζονται από διαφορετικές συσκευές.
- Κάθε τουρνουά = ένα Redis hash με ξεχωριστό πεδίο ανά παίκτη → οι εγγραφές/υποβολές γίνονται **ατομικά** (δεν «πατάει» ο ένας παίκτης τον άλλον).
- Στην οθόνη του draft εμφανίζεται από την αρχή **κωδικός & σύνδεσμος πρόσκλησης** (κουμπιά «Copy code» / «Copy link»), για να μπαίνουν φίλοι όσο φτιάχνεις το ρόστερ.
- Νέα αρχεία: `index.html` (παιχνίδι), `api/tournament.js` (serverless συνάρτηση), `DEPLOY.md` (οδηγίες deploy).

## 🔧 Διορθώσεις
- **Σύνθετες ομάδες «Α;Β»** (παίκτες που άλλαξαν ομάδα μέσα στη σεζόν) διαχωρίστηκαν — εμφανίζονται στις πραγματικές τους ομάδες, όχι σε ψεύτικη «διπλή» ομάδα.
- Εμφανίζονται πλέον παίκτες με **≥15 αγώνες** (πριν ≥20) → λιγότεροι «κρυμμένοι» παίκτες, ιδίως σε παλιότερες/νέες ομάδες.
- Ο σύνδεσμος πρόσκλησης δείχνει στο **public domain** και όχι σε κλειδωμένο preview URL του Vercel.

## 🚀 Deploy
Πλήρεις οδηγίες στο `DEPLOY.md`. Το `index.html` σερβίρεται από τη ρίζα στο Vercel· για το multiplayer χρειάζεται συνδεδεμένο **Vercel KV (Upstash)** store (το Vercel βάζει αυτόματα τα `KV_REST_API_URL` / `KV_REST_API_TOKEN`).

Σημειώσεις:
- Ο σύνδεσμος πρόσκλησης είναι ρυθμισμένος για το deployment `euroleague-guru.vercel.app`· για άλλο domain αλλάζει σε μία γραμμή (`getTournLink`).
- Τα `eurodraft_10/11/12.html` είναι ενδιάμεσα στιγμιότυπα της εξέλιξης· η ζωντανή έκδοση είναι το `index.html`.
- Οι θέσεις κάποιων νέων παικτών της 2025-26 (που δεν είχε η Wikidata) μπήκαν χειροκίνητα κατά προσέγγιση και διορθώνονται εύκολα.
