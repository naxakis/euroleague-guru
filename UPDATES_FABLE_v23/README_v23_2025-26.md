# GuruLeague — Ενσωμάτωση σεζόν 2025-26 στο ERA DB (v23)
*(06/07/2026 — βασισμένο στο Database_2025-2026.xlsx + το τρέχον index του repo)*

## Τι έγινε
Η σεζόν 2025-26 (341 γραμμές, 20 σύλλογοι) εντάχθηκε στο era **2020-2025** ως εξής:

1. **125 συγχωνεύσεις** σε υπάρχοντες παίκτες (ίδιος σύλλογος): σταθμισμένοι μέσοι όροι με
   βάρος τα GP, νέο συνολικό GP, `seasons: "2020-2026"`, `nSeasons +1`.
   (π.χ. Vezenkov: 147gp/15.9 → 186gp/16.6)
2. **Οι 119 standalone εγγραφές 2025-26 που υπήρχαν ήδη ΞΑΝΑΧΤΙΣΤΗΚΑΝ** από το αρχείο,
   γιατί είχαν ένα συστηματικό λάθος: στους παίκτες με 2 συλλόγους είχε περαστεί το
   ΣΥΝΟΛΟ σεζόν και στους δύο (π.χ. Saben Lee 27gp/8.0 και σε Efes και σε ΟΣΦΠ — σωστό:
   16gp/11.3 Efes + 11gp/3.4 ΟΣΦΠ· ομοίως Bruno Fernando, Tyrique Jones).
   Οι θέσεις (pos/cpos) που είχες βάλει διατηρήθηκαν.
3. **170 νέες εγγραφές** (GP≥10) με θέσεις από: τις δικές σου εγγραφές → παλιότερες
   εγγραφές του παίκτη σε άλλη ομάδα → χειροκίνητη απόδοση (~39 ολοκαίνουργιοι, π.χ.
   Dinwiddie PG, Holmes C, Alex Len C, Cameron Payne PG).
4. **Ενοποίηση ονομάτων (canonical)**: όλα τα variants του αρχείου δέθηκαν στα ονόματα
   του DB — Nigel Hayes → **Nigel Hayes-Davis** (νέα εγγραφή στον ΠΑΟ, 16gp/14.1),
   Daniel/Dan Oturu, Matt(hew) Costello, Nick Calathes, Nikos Rogkavopoulos,
   Brandon Boston, Shake (Malik) Milton, Chris Silva κ.ά. Μηδέν διπλότυπα εντός ομάδας
   (ελέγχθηκε αυτόματα).

## Πρωταθλητές (για το Gauntlet)
- Νέο `champ_keys_v3.json`: κλειδιά-ονόματα όλων των πρωταθλητών EL της βάσης
  (ρόστερ 2006-2024 από db_year + patch προ-2005 + **Fenerbahce 2025** + **Olympiacos 2026**

  Vezenkov, Walkup, Dorsey, Fournier, Milutinov, Papanikolaou, Peters, McKissic,
  Ntilikina, Donta Hall, Tyrique Jones, Saben Lee, Tyson Ward.
- Το gauntlet_demo.html χρησιμοποιεί ήδη τη v3.

## Προσοχή / εκκρεμότητες
- **PIR (value)**: το αρχείο της σεζόν δεν έχει PIR → οι 170 νέες εγγραφές έχουν value 0
  (σύνολο 138 εγγραφές με value 0 στο era). Αν το equation engine (teamPower δεν το
  χρησιμοποιεί, αλλά ο Guru του demo ναι) βασιστεί στο value, οι rookies υποτιμούνται.
  Όποτε βρεθεί πηγή PIR 2025-26, γίνεται ένα πέρασμα ενημέρωσης.
- Το era κρατά το key **"2020-2025"** (δεν άλλαξαν keys/ERAS στο UI) — οι εγγραφές μέσα
  δηλώνουν "2020-2026"/"2025-2026". Αν θέλετε rename σε "2020-2026" στη ρόδα, είναι
  αλλαγή ενός key + όσων UI strings το δείχνουν.
- CSKA/Zenit/Khimki έμειναν ως έχουν (δεν έπαιξαν 2025-26).

## Αρχεία
- `index_v23_with_2025-26.html` → αντικαθιστά το index (περιέχει ΚΑΙ τις αλλαγές v22:
  spin easing, SEO, gp-guard). Αν έχεις ήδη κάνει deploy το v22, το diff είναι μόνο το DB.
- `db_era_2020-2026.json` → μόνο το DB object, αν προτιμάς να κάνεις το swap μόνος σου.
- `champ_keys_v3.json` → δεδομένα πρωταθλητών για το Gauntlet.
- `gauntlet_demo.html` → το demo με τη νέα βάση (νέο: ο Hayes-Davis νάρκη και στον ΠΑΟ).
